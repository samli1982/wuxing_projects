### Step 2: 快速替换（每个页面 WXSS）

使用查找替换：

```
[查找] #BBBBBB      →  [替换] #9CA3AF
[查找] #FFFFFF      →  [替换] #D1D5DB  
[查找] #E6E6E6      →  [替换] #D1D5DB
[查找] rgba(255,255,255,0.1)   →  [替换] rgba(58, 158, 143, 0.15)
```

### Step 3: 更新卡片样式

**旧写法：**
```css
background: rgba(255,255,255,0.06);
border: 1px solid rgba(255,255,255,0.1);
```

**新写法：**
```css
background: rgba(255, 255, 255, 0.06);
backdrop-filter: blur(10px);
border: 1px solid rgba(58, 158, 143, 0.15);
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
```

### Step 4: 增强动效

**旧写法：**
```css
transition: all 0.3s;
transform: scale(0.95);
```

**新写法：**
```css
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
transform: scale(0.98);
box-shadow: 0 1px 4px rgba(58, 158, 143, 0.15);
```

## 📊 设计原则

### ✨ 五大支柱

1. **深空意境** 
   - 全局背景 `#1A1C29` 营造宇宙感
   - 保持暗色系一致性

2. **科技禅意**
   - 磨砂玻璃效果 + 超低透明度
   - 减少视觉噪点，提升克制感

3. **水墨美学**
   - 渐变分割线替代实线
   - 东方留白意境体现

4. **信息优先**
   - 精准的文字色彩层级
   - 用对比度引导视线
   - `#D1D5DB` (主) > `#9CA3AF` (次) > `#6B7280` (辅)

5. **交互克制**
   - 所有动效 `0.3s + cubic-bezier(0.4, 0, 0.2, 1)`
   - 防止过度装饰
   - 缩放 `scale(0.98)` 而非 `0.95`

### 💡 设计理念

> "这不是一个普通的健康APP，而是一本会呼吸的《天人之书》。"

## ⚠️ 重要注意事项

### ❌ 停用的颜色
- `#BBBBBB` ❌ 过于偏黄，显"脏"
- `#E6E6E6` ❌ 纯粹且刺眼
- `#A0A0A0` ❌ 缺乏温度
- 褐色、暗橙、棕色 ❌ 脏感

### ✅ 必须使用
- **主文字**: `#D1D5DB` (不可替代)
- **次文字**: `#9CA3AF` (不可替代)
- **强调**: `#3A9E8F` (绿色圆润感)
- **背景**: `#1A1C29` (深空蓝)

### 📏 间距标准
- 所有间距必须是 **8 的倍数**
- 推荐：8px, 12px, 16px, 20px, 24px, 32px

### 🎯 圆角标准
- 卡片/按钮/输入框: **12px**
- 标签/徽章: **6px**
- 小元素: **4px**
