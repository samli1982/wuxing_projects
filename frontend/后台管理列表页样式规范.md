# 后台管理列表页标准样式规范

## 布局结构

```
页面容器 → 卡片 → 卡片头部 + 卡片内容（搜索表单 + 表格 + 分页）
```

## 关键样式参数

### 1. 页面容器 (`.xxx-management`)

```css
.xxx-management {
  padding: 10px;
  height: calc(100vh - 60px);
  box-sizing: border-box;
  display: flex;
  flex-direction: column;
}
```

### 2. 卡片 (`.el-card`)

```css
.xxx-management .el-card {
  flex: 1;
  margin-bottom: 10px;  /* 卡片与页面底部间距 */
  overflow: hidden;
  display: flex;
  flex-direction: column;
}
```

### 3. 卡片头部 (`.el-card__header`)

```css
.xxx-management .el-card :deep(.el-card__header) {
  padding: 9px 10px;
  border-bottom: none;  /* 无分隔线 */
}
```

### 4. 卡片内容 (`.el-card__body`)

```css
.xxx-management .el-card :deep(.el-card__body) {
  flex: 1;
  display: flex;
  flex-direction: column;
  overflow: hidden;
  padding: 10px 10px 0 !important;  /* 底部无padding */
}
```

### 5. 搜索表单 (`.search-form`)

```css
.search-form {
  margin-top: 10px;     /* 与头部间距 */
  margin-bottom: 10px;
}
```

### 6. 表格 (`.el-table`)

```css
.xxx-management .el-table {
  flex: 1;  /* 占满剩余空间 */
}
```

### 7. 分页容器 (`.pagination-container`)

```css
.pagination-container {
  margin-top: 5px !important;
  padding: 0 !important;
  border-top: none !important;     /* 无分隔线 */
  display: flex;
  justify-content: flex-end;       /* 右对齐 */
}

.pagination-container :deep(.el-pagination) {
  padding: 0 !important;
  margin: 0 !important;
}
```

### 8. 分页组件配置

```vue
<el-pagination
  v-model:current-page="pagination.current"
  v-model:page-size="pagination.size"
  :total="pagination.total"
  :page-sizes="[20, 30, 50, 100]"
  layout="total, sizes, prev, pager, next, jumper"
  @size-change="handleSizeChange"
  @current-change="handleCurrentChange"
/>
```

```javascript
const pagination = reactive({
  current: 1,
  size: 20,    // 默认每页20条
  total: 0
})
```

## 设计特点

- ✅ **紧凑布局**：最大化表格显示空间
- ✅ **无冗余分隔线**：仅保留必要的视觉分隔
- ✅ **卡片底部间距**：与页面底部保持10px间距
- ✅ **极简分页区域**：高度极小，无多余padding
- ✅ **统一间距体系**：10px/5px体系
